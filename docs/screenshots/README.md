# Screenshots

Drop PNG screenshots here using these **exact filenames** — they are referenced
from the main [`README.md`](../../README.md) and will render automatically once
committed.

Recommended: ~1600 px wide, PNG, no personal data / secrets visible. Light or dark
theme is fine.

| Filename | Shows |
|---|---|
| `setup.png` | First-time setup (create the admin account) |
| `dashboard.png` | Dashboard / overview |
| `endpoints.png` | Endpoints management |

To add or replace a screenshot: copy the file here with the matching name, then
`git add docs/screenshots/<name>.png` and push.
